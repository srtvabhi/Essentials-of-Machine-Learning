# 📘 Machine Learning (ML) 

## 1. What is Machine Learning (ML)?
Machine Learning is a part of AI where computers learn from data and improve automatically without being explicitly programmed.

### Real-life Use Cases:
- Netflix recommends movies
- Google Maps predicts traffic

---

## 2. Types of Learning

### Supervised Learning
- Uses labeled data (input + output)

#### Use Case:
- Spam detection
- House price prediction

### Unsupervised Learning
- Uses unlabeled data

#### Use Case:
- Customer segmentation
- Market basket analysis

---

## 3. Supervised Learning Types

### Regression
- Predicts continuous values

#### Use Case:
- Salary prediction
- Temperature prediction

### Classification
- Predicts categories

#### Use Case:
- Disease detection
- Loan approval

---

## 4. Regression Concepts

Formula:
y = mx + b

- x → input
- y → output
- m → slope
- b → bias

Training Data → used to train  
Testing Data → used to test  

---

## 5. Classification

- Assigns data to categories

Types:
- Binary
- Multi-class

---

## 6. Regression Metrics

### MAE
MAE = (1/n) Σ |y - ŷ|

### MSE
MSE = (1/n) Σ (y - ŷ)^2

### RMSE
RMSE = sqrt((1/n) Σ (y - ŷ)^2)

---

## 7. Classification Metrics

### Accuracy
Accuracy = correct / total

### Precision
Precision = TP / (TP + FP)

### Recall
Recall = TP / (TP + FN)

### F1 Score
F1 = 2 * (Precision * Recall) / (Precision + Recall)
